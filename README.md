# zhihuinl-frontend
智慧农林网 前台&后台管理系统
当时计划要写网站首页和网站后台管理系统，写到管理后台的时候，不想重开一个项目（其实就是懒），于是做了下鉴权，普通用户登录会进到网站首页，管理员用户登录会进到后台管理系统。

## 技术栈
用了Ant Design Pro，省去了项目搭建的时间。
React + umi + dva + antd

## 相关命令

安装依赖包:

```bash
npm install
```

或者

```bash
yarn
```

### 运行项目

```bash
npm start
```

### 打包项目

```bash
npm run build
```
